Comparison with other software libraries

Here you can compare the performance of the algorithms in pyDML with the performances of other popular distance metric learning software libraries. Some of the algorithms shared among these libraries have been compared. Currently, the following libraries are being compared:

  • metric-learn: Metric Learning in Python. A Python library with several supervised and weak-supervised metric learning algorithms.
  • DistLearn Kit: A Matlab toolkit for distance metric learning. A collection of several implementations of distance metric learning algorithms.
