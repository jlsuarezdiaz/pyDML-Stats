Kernel Experiments.

Here you can see the results of the kernel-based algorithms, using the 3-NN score as metric. The used kernels are:

  • Linear: Linear kernel.
  • Poly-2: Polynomial kernel of degree 2.
  • Poly-3: Polynomial kernel of degree 3.
  • RBF: Gaussian kernel (radial basis function).
  • Laplacian: Laplacian kernel.
See the Scikit-Learn kernel documentation for more details.
